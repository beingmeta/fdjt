HatchShow/FD is a portable typesetting plugin. It balances measures. It was inspired by a JQUERY implementation from Charlie Park.

The “measure” is the width of a column of text.

(↑this is a measure)

Each line in this code’s HTML is set off with a <span> element, & HatchShow adjusts the font size automatically. Each line should end up (roughly) the same width.

This text will expand or contract, depending on how many characters are in the line.

It’s named after the classic print shop Hatch Show Print

Every font size on this page was generated by an algorithm.

HatchShow/FD can be used by itself or as part of the FDJT javascript toolkit developed and maintained by beingmeta

You can use HatchShow/FD by referencing or including the javascript source and adding the “hatchshow” class to or around the elements which you wish to be adjusted.
